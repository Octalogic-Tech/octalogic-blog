## Getting Started

1. Install the doppler cli [https://docs.doppler.com/docs/install-cli](https://docs.doppler.com/docs/install-cli#installation)
2. Run `doppler setup --token=TOKEN --no-interactive`
3. Run `npm run local:dev`
